Code Snippet: iRobot Creates in Jython

  • Running a Create agent with a robot: CreateIntroSession.py
  • An experiment about nexting on the Create with the cliff sensors: CreateNexting.py

Dependencies

zephyr.plugin.core.api , rlpark.plugin.rltoys , rlpark.plugin.robot , rlpark.plugin.irobot

How to run this code

  • Start Zephyr with RLPark plugins installed
  • File->Open File... will start running the file
Note: it can also be run directly with Jython (without Zephyr) if RLPark is in the Jython classpath.
